## Про проект

Простой демонстрационный проект по базовому управлению тикетами (напоминалками) с напоминалкой "Просрочено".

Включает:
 - репозитории
 - разделение логики
 - работа с моделями и уровнями доступа DTO
 - оформление кода
 - базовая валидация
 - реализация доступа через фасады

#База
Laravel 7.15, php7.3
